Manual de preinscripción para aspirantes
========================================

Objetivo
--------
Este instructivo describe cada pantalla del asistente de preinscripción online del IPES Paulo Freire y los datos que debés cargar para completar el trámite correctamente.

Requisitos previos
------------------
- Verificar que exista una **ventana de preinscripción** activa (la portada del sitio muestra un aviso en caso contrario).
- Contar con un correo electrónico válido y un número de celular actualizado.
- Digitalizar en PDF/JPG/PNG la documentación obligatoria (DNI frente/dorso, título o constancia, certificados requeridos, foto 4x4).

Ingreso al asistente
--------------------
1. Ingresá a la URL oficial y elegí “Preinscripción”.
2. El sistema muestra una pantalla inicial con el botón `Comenzar`. Si está habilitada la protección, primero deberás resolver el reCAPTCHA.
3. Cada paso del asistente guarda el progreso en tu navegador; podés salir y volver mientras la ventana continúe abierta.

Pasos del asistente
-------------------

1. Datos personales
~~~~~~~~~~~~~~~~~~
- Campos obligatorios: apellido/s, nombre/s, DNI, CUIL, fecha y lugar de nacimiento, género, nacionalidad.
- Verificá que coincidan con tu DNI. El sistema valida formato de DNI y fechas antes de continuar.

2. Contacto y domicilio
~~~~~~~~~~~~~~~~~~~~~~~
- Completa domicilio real (calle, número, barrio, localidad, provincia, país, código postal).
- Añade correo electrónico, teléfono celular, teléfono alternativo y un contacto de emergencia (nombre, parentesco, teléfono).
- Estos datos se usarán para notificaciones automáticas, por lo que deben mantenerse vigentes.

3. Estudios previos
~~~~~~~~~~~~~~~~~~~
- **Secundario:** institución, localidad, provincia, título obtenido y fecha de egreso.
- **Superiores (opcional):** si contás con estudios terciarios/universitarios, registralos indicando título y estado.
- Si todavía no disponés del analítico definitivo, marcá la casilla “Presentaré constancia” para informar a Secretaría.

4. Carrera y documentación
~~~~~~~~~~~~~~~~~~~~~~~~~~
- Seleccioná el profesorado/carrera disponible. Si la ventana habilita varias opciones, deberás elegir una antes de avanzar.
- Carga la foto 4x4 (JPEG/PNG, hasta 1.5 MB). El sistema muestra una vista previa; si no se ve, reintenta.
- Marcá en el checklist qué documentación ya tenés lista para presentar presencialmente (fotocopia DNI, certificado de salud, etc.). Esto ayuda a Bedelía a planificar la formalización.

5. Declaración jurada y confirmación
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
- Se muestra un resumen editable. Podés regresar a cualquier paso con el botón “Volver”.
- Lee la declaración jurada y marca la casilla de conformidad.
- Presioná `Descargar PDF` para generar la planilla; el botón final se habilita recién después de la descarga.
- Finalmente hacé clic en `Enviar preinscripción`. Si el proceso fue exitoso verás un mensaje con tu código `PRE-XXXX`.

Después del envío
-----------------
- Conservá el PDF y el código `PRE-XXXX`. Servirán para consultas y para la instancia presencial.
- Si proporcionaste un correo válido recibirás una confirmación automática (revisá la carpeta de spam).
- Mientras la ventana siga abierta podés reingresar con tu DNI y código para actualizar datos o descargar nuevamente el formulario.

Formalización presencial
------------------------
1. Presentate en Secretaría en las fechas comunicadas.
2. Lleva la planilla impresa, los originales y copias de la documentación detallada en el checklist, y los folios solicitados.
3. El personal validará los datos en el panel `Confirmar Inscripción`, registrará la documentación recibida y cambiará tu estado a **Confirmada** u **Observada** según corresponda.

Seguimiento y soporte
---------------------
- Podés consultar el estado de tu trámite ingresando nuevamente al asistente o, una vez creada tu cuenta de alumno, desde el portal `/alumnos`.
- Si la solicitud queda “Observada”, revisá el mensaje asociado para saber qué documentación falta y presentala cuanto antes.
- Ante problemas técnicos (captcha, carga de archivos, errores de servidor) envía un correo a soporte institucional adjuntando captura y descripción del paso en el que ocurrió.

Buenas prácticas
----------------
- Guardá una copia digital de toda la documentación en un servicio en la nube para disponerla si el sistema la solicita nuevamente.
- Cargá los datos con calma; los campos obligatorios están marcados con * y el asistente no permite continuar si falta información.
- Si cambiás de correo o teléfono luego de enviar la solicitud, informalo durante la formalización presencial para que puedan actualizar tu legajo.
