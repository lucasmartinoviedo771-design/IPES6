# Manual de Usuario - Sistema de Gestión IPES6

Bienvenido a la documentación del Sistema Integral de Gestión de Estudiantes del I.P.E.S. "Paulo Freire".

## Manuales por Rol

A continuación, encontrará las guías de uso específicas para cada perfil de usuario en el sistema:

*   **[Manual para Estudiantes](manuales/manual_estudiante.md)**: Guía para preinscripción, inscripción a materias, consulta de notas y regularidades.
*   **[Manual para Docentes/Tutores](manuales/manual_tutor.md)**: Gestión de cursos, carga de notas y asistencia.
*   **[Manual para Bedeles](manuales/manual_bedel.md)**: Administración diaria, gestión de asistencias y novedades.
*   **[Manual para Coordinadores](manuales/manual_coordinador.md)**: Supervisión académica y gestión de carreras.
*   **[Manual para Secretaría](manuales/manual_secretaria.md)**: Gestión administrativa, legajos y documentación.
*   **[Manual para Jefes de Departamento](manuales/manual_jefes.md)**: Gestión de departamentos y asignación docente.
*   **[Manual de Administración](manuales/manual_admin.md)**: Configuración del sistema, usuarios y permisos.
*   **[Manual de Inscripción](manuales/manual_inscripcion.md)**: Guía detallada del proceso de inscripción.
