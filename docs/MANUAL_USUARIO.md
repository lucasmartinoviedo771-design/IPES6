# Manual de Usuario consolidado - Sistema IPES6

Este documento unifica las guías de uso de todos los perfiles en el sistema, desde estudiantes hasta administradores.

## 1. Guía para Estudiantes e Inscripciones

- **Acceso**: Se realiza con DNI como usuario y contraseña (inicialmente).
- **Preinscripción**: Formularios guiados paso a paso con carga de documentación.
- **Inscripción a Materias**: El alumno puede inscribirse a materias según su plan de estudios, consultando correlatividades y horarios de comisiones disponibles.

## 2. Guía para Secretaría y Administración

- **Legajos**: Gestión completa de datos personales y académicos de alumnos.
- **Planes de Estudios**: Creación y edición de profesorados, materias y correlatividades.
- **Mantenimiento**: Bedeles cargan asistencias y regularidades.

## 3. Guía de Administración General

- **Ventanas de Habilitación**: Administrador abre/cierra los periodos de preinscripción e inscripción en el sistema (el frontend bloqueará el acceso fuera de estas fechas).
- **Usuarios y Roles**: Gestión de altas/bajas de personal y asignación de permisos (ver `docs/DOCUMENTACION_TECNICA.md` para la matriz detallada).

## 4. Modelos de Comunicación (Mails)

**Modelo: Bienvenido al Sistema IPES6**
```text
Asunto: Tu preinscripción en el IPES "Paulo Freire" ha sido recibida.

Hola [Nombre],
Tu solicitud para la carrera [Carrera] ha sido registrada con éxito.
Para completar el trámite, preséntate en bedelía con la siguiente documentación original:
- DNI original y copia.
- Título secundario / Certificado de título en trámite.
- Dos fotos carnet.
- Carpeta colgante (tamaño oficio).

¡Te esperamos!
IPES Paulo Freire.
```
